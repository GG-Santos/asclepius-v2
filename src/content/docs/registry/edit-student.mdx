Edit student records before graduation whenever the correction belongs to the
training record: name, photo, batch assignment, quiz scores, practical scores,
or training status.

<OpenRoute href="/dashboard/students">Open Students</OpenRoute>

<WorkflowMedia title="Edit a Student" slug="edit-student" />

## What You Can Change

| Area | Notes |
| --- | --- |
| Profile | First, Middle, Last, Suffix, and Batch. |
| Photo | Replace or edit the portrait used later on artifacts. |
| Quizzes | Raw quiz scores that roll up to SJE. |
| Practicals | Raw FWE, EP, PAS, CCST, and CCSM scores. |
| Actions | Save, Graduate one student, or Delete the student. |

## Steps

<Steps>

<Step>

### Find the student

Open Students and search by name or enrollment number. Use the edit action on
the target row.

</Step>

<Step>

### Review the whole record

Check Photo, Profile, Quizzes, and Practicals before changing anything. A score
fix can affect graduation outcome and ranking.

</Step>

<Step>

### Apply the correction

Change the smallest set of fields that matches the source evidence.

</Step>

<Step>

### Save changes

Choose **Save changes**, then confirm the save dialog. The app returns to the
Students list.

</Step>

<Step>

### Reopen if needed

For score corrections, reopen the record or the batch graduation dialog to
confirm the total and graduation preview are now correct.

</Step>

</Steps>

## Delete From Student Edit

The student edit form also has **Delete**. Use it only for duplicate or mistaken
training records. Deleting a student removes the training record permanently.

## Troubleshooting

| Symptom | Cause | Fix |
| --- | --- | --- |
| Save asks for confirmation | Saves are intentionally confirm-before-apply. | Read the dialog and choose **Save changes** if correct. |
| The Graduate button is visible | The record can be promoted individually. | Prefer batch graduation unless you have an approved one-off reason. |
| Score rows changed after selecting another batch | The selected batch controls grading setup. | Confirm the intended batch before saving. |
| Delete is too risky | The record may be real training history. | Leave it in place and correct status or batch assignment instead. |

<DocNote title="Fix source data before graduation" tone="warn">
Once a student becomes a graduate, public credential data moves to the graduate
registry. Correct in-training issues while the record is still a student.
</DocNote>
