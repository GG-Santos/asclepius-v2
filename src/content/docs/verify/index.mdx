---
title: Verify a credential
description: Confirm an EMT's WSL EMS license in seconds — no account, no cost.
---

Anyone can check an EMT's credential against the WSL EMS registry. There's no
account and no cost, and the result reflects the same record the issuing
institution holds.

## Run a verification

<Steps>

<Step>

### Find the License Number

The **LCN** is printed on the EMT's card and certificate, formatted like
`A09-240801`.

</Step>

<Step>

### Enter it on the home page

Type the LCN into the search box and press **Verify License**. Enter it exactly
as printed — surrounding spaces are ignored, but the rest is matched exactly
(including capitalization).

</Step>

<Step>

### Read the result

A match opens the credential page; anything else returns a clear "no active
credential" result.

</Step>

</Steps>

## What a result includes

When a credential matches, the page shows:

- **Profile** — the licensee's name, and photo if one is on file.
- **Proficiency record** — module scores (not shown for [Legacy](/docs/verify/statuses) records).
- **License card** — the official card, viewable full-size.
- **Certificate** — the certificate of completion.
- **QR code** — links back to this exact page. See [Verification links & QR codes](/docs/verify/sharing).

## If nothing matches

A number that isn't in the registry — or a credential that has been archived —
returns **"No Active Credential Found."** Re-check the characters before
concluding the credential isn't real; see [Troubleshooting](/docs/troubleshooting).

<Callout type="info" title="The result is the source of record">
A verification page reflects the registry directly. If a credential doesn't
appear under its number, WSL EMS has no active record of it.
</Callout>
